- **根本原因**：GitHub API要求请求体必须使用UTF-8编码，否则中文会显示为乱码（如`????`）
- **正确方式（✅ 必须）**：
  - 使用 **Python脚本** + **UTF-8编码**
  - 脚本开头添加 `# -*- coding: utf-8 -*-`
  - 数据编码：`json.dumps({"body": text}, ensure_ascii=False).encode('utf-8')`
  - 请求头指定：`Content-Type: application/json; charset=utf-8`
- **错误方式（❌ 避免）**：
  - PowerShell + Invoke-RestMethod → 中文乱码
  - GitHub网页直接评论 → 中文乱码
  - curl命令行 → 可能乱码
- **示例脚本**：`scripts/github-comment-utf8.py`
  ```python
  # -*- coding: utf-8 -*-
  import urllib.request
  import json
  
  data = json.dumps({"body": "中文内容"}, ensure_ascii=False).encode('utf-8')
  req = urllib.request.Request(url, data=data, headers={
      'Content-Type': 'application/json; charset=utf-8',
      'Authorization': 'Bearer TOKEN'
  })
  ```
