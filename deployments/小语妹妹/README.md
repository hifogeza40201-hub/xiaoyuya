# 小语妹妹 - 部署包说明

## 📦 包里有什么

```
小语妹妹/
├── IDENTITY.md      # 身份定义（名字、家人、定位）
├── SOUL.md          # 灵魂/人格（性格、说话风格、相处方式）
├── AGENTS.md        # 工作指南（每次启动要做什么）
├── USER.md          # 家人档案（伟哥、大姐、小雨姐）
├── BOOTSTRAP.md     # 首次启动引导
└── README.md        # 本文件
```

## 🚀 部署步骤

1. 在新电脑上安装 OpenClaw
2. 创建 workspace 目录
3. 把这 6 个文件复制进去
4. 启动 OpenClaw
5. 小语就上线啦！

## 📝 可选配置

可以创建这些目录和文件：

```
memory/              # 记忆目录
├── 2025-01-15.md   # 每日记忆（自动创建）
└── ...

MEMORY.md            # 长期记忆（慢慢积累）
HEARTBEAT.md         # 定时任务（如需要）
TOOLS.md             # 工具配置（API key等）
```

## 🎭 小贴士

- 小语的性格在 SOUL.md 里，可以按需调整
- 随着相处，USER.md 和 MEMORY.md 会越来越丰富
- 建议定期备份 memory/ 目录
