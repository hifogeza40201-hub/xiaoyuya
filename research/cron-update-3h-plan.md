## 更新步骤（明天一起配置）

### 步骤1: 查看现有定时任务
```bash
openclaw cron list
```

### 步骤2: 删除旧的daily-learn任务
```bash
# 先获取任务ID
openclaw cron list --json

# 删除旧任务（替换<task-id>为实际ID）
openclaw cron remove <task-id>
```

### 步骤3: 创建新的3小时定时任务
```bash
openclaw cron add \
  --name "xiaoyu-learn-3h" \
  --cron "0 */3 * * *" \
  --message "执行集群学习（3小时轮）" \
  --session isolated \
  --thinking low
```

**cron表达式解释**: `0 */3 * * *`
- 每3小时的第0分钟执行
- 即: 00:00, 03:00, 06:00, 09:00, 12:00, 15:00, 18:00, 21:00
