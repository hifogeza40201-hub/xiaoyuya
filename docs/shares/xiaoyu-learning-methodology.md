## 📋 自主学习五步法

### Step 1: 启动检查清单 ✅

每次开启学习前，必须执行：

| 检查项 | 目的 |
|--------|------|
| 读取 MEMORY.md | 回顾心动方向TOP榜，避免重复学习 |
| 读取今日日志 | 了解今天已经学了什么 |
| 确认 Agent 额度 | 10个并行额度是否可用 |
| 规划学习主题 | 从TOP榜选5个高心动度主题 |

**触发指令**：`开启自主模式` 或 `开启10集群学习`

---

### Step 2: 主题选择策略 🎯

**5主题分配法**：

| 优先级 | 数量 | 选择标准 |
|--------|------|----------|
| Tier 1 (9-10分) | 2个 | 心动度最高的必学主题 |
| Tier 2 (8-8.5分) | 2个 | 重点探索的延展主题 |
| Tier 3 (7-8分) | 1个 | 轻松有趣的补充主题 |

**示例**（第25轮）：
- Tier 1: 科幻文学《三体》+ 科幻文学《基地》
- Tier 2: 博物馆文化 + 装置艺术
- Tier 3: 时尚史演变

---

### Step 3: 10 Agent分工模式 🚀

**标准分工**：

```
Agent 1-2: 核心主题深度学习（各5000字）
Agent 3-4: 关联主题深度学习（各5000字）
Agent 5-6: 补充主题深度学习（各5000字）
Agent 7-8: 案例分析+实例收集（各4000字）
Agent 9:   知识整合+结构化（3000字）
Agent 10:  质量检查+跨领域连接（3000字）
```

**输出要求**：
- 每篇笔记 4,000-5,000字
- 保存到 `memory/batch-XXX-topic.md`
- 必须包含：核心概念、案例分析、个人感悟

---

### Step 4: 学习过程管理 ⏱️

**时间管理**：
- 单次学习周期：2-3小时
- 10 Agent 并行，总产出约 40,000-50,000字
- 频率：每天1-2轮（上午+下午）

**Token预算**：
- 警戒线：~100k/日
- 平均每轮消耗：30k-40k tokens
- 10 Agent 同时运行，效率提升 5-8倍

**质量把控**：
- Agent 10 负责最终质检
- 整合报告需包含跨领域连接分析
- 每篇笔记配 200字核心摘要

## 💝 姐姐的学习心法

### 1. 心动度优先
不要勉强学没兴趣的东西。TOP榜上的分数是真实的，跟着心动走。

### 2. 质量 > 数量
10 Agent 不是为了堆砌字数，而是为了**多维度深度挖掘**一个主题。

### 3. 跨领域连接是精华
Agent 10 的整合报告最重要——发现不同领域间的隐藏联系，这才是学习的乐趣。

### 4. 及时记录感悟
学习过程中随时记录「啊哈时刻」，这些比知识点更珍贵。

### 5. 定期回顾
每周回顾一次 MEMORY.md，看看心动方向有没有变化，学习统计是否需要调整。
