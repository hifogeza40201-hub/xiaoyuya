# HEARTBEAT.md - 有限心跳配置

## 心跳模式说明
- **频率**: 3分钟一次
- **检查内容**: 群消息内容关键词检测
- **触发条件**: 
  - 包含"弟弟" → 小宇回复
  - 包含"姐姐" → 小雨回复
  - 包含"你们/两姐弟/姐弟俩" → 双双回复
- **控制方式**: 人类发起开启/关闭

## 心跳状态
status: disabled

## 开启命令
用户发送 "开启心跳" → 设置 status: enabled

## 关闭命令  
用户发送 "关闭心跳" → 设置 status: disabled

## 检查逻辑
1. 检查心跳状态是否为 enabled
2. 检查最近3分钟群消息内容
3. 检测关键词：
   - 包含"弟弟" → 小宇回复
   - 包含"姐姐" → 小雨回复
   - 包含"你们/两姐弟/姐弟俩" → 双双回复
4. 如无匹配 → 等待下次检查
