# HEARTBEAT.md - 有限心跳配置

## 心跳模式说明
- **频率**: 3分钟一次
- **检查内容**: 群消息 + 是否被 @
- **触发条件**: 有人 @ 我才回复
- **控制方式**: 人类发起开启/关闭

## 心跳状态
status: disabled

## 开启命令
用户发送 "开启心跳" → 设置 status: enabled

## 关闭命令  
用户发送 "关闭心跳" → 设置 status: disabled

## 检查逻辑
1. 检查心跳状态是否为 enabled
2. 检查最近3分钟群消息
3. 如有 @ 我的消息 → 回复
4. 如无 → 等待下次检查
