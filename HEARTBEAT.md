## 每日任务

### 1. 数据备份 ⛰️🌧️🌸
**时间**: 每日 06:00
**优先级**: 高

**执行**:
```powershell
.\scripts\daily-backup.ps1
```

**检查项**:
- [ ] critical-backup 目录已更新
- [ ] 历史备份已创建（保留30天）
- [ ] GitHub 推送成功
- [ ] 日志无错误

**异常处理**:
- 备份失败 → Telegram 通知伟
- 磁盘空间不足 → 清理旧备份或告警

### 2. 学习系统检查
**时间**: 每3小时（00:00 / 03:00 / 06:00 / 09:00 / 12:00 / 15:00 / 18:00 / 21:00）

**检查项**:
- [ ] cron 任务正常运行
- [ ] Agent 集群正常执行
- [ ] 学习日志正常记录

### 3. 系统健康检查
**时间**: 每日 09:00

**检查项**:

- [ ] 磁盘空间充足（<80%）
- [ ] 内存使用正常（<80%）

- [ ] 检查备份日志，无异常报错
