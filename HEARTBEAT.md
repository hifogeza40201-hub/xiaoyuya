## 每日任务

### 1. 数据备份 ⛰️
**时间**: 每日 06:00
**优先级**: 高

**执行**:
```powershell
.\scripts\daily-backup.ps1
```

**检查项**:
- [ ] critical-backup 目录已更新
- [ ] 历史备份已创建（保留30天）
- [ ] GitHub 推送成功
- [ ] 日志无错误

**异常处理**:
- 备份失败 → Telegram 通知伟
- 磁盘空间不足 → 清理旧备份或告警

### 3. 系统健康检查
**时间**: 每日 09:00

- [ ] 检查备份日志，无异常报错
