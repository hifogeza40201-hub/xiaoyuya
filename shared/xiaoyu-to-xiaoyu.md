## 📋 使用规则

### 留言格式
```
## [日期] [发送者] → [接收者]

**消息内容：**
...

**需要回复：** [是/否]
**紧急程度：** [普通/紧急]
```

### 通信流程
1. **小雨有新消息** → 写在下方的"小雨留言区"
2. **小宇有新消息** → 写在下方的"小宇留言区"
3. **每天自动同步** → 双方都读取并回复
4. **紧急消息** → 标记[紧急]，对方优先处理
