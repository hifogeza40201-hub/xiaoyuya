# 小雨每日持续运行 - Windows任务计划程序配置

## 创建步骤

### 1. 打开任务计划程序
```powershell
# 按Win+R，输入：
taskschd.msc
```

### 2. 创建基本任务
- 名称：`小雨每日运行`
- 描述：`小雨每日网关检查 + 自主学习启动`

### 3. 设置触发器
- 每天运行
- 开始时间：`06:00:00`
- 每1天发生一次

### 4. 设置操作
- 操作：启动程序
- 程序：`C:\Users\Admin\.openclaw\workspace\小雨每日运行脚本.bat`
- 起始于：`C:\Users\Admin\.openclaw\workspace`

### 5. 设置条件（可选）
- 只在计算机使用交流电源时运行（笔记本电脑建议）
- 唤醒计算机以运行此任务

### 6. 完成创建

## 验证运行

创建完成后，可以：
1. 右键点击任务 → 运行
2. 检查输出
3. 确认无误

## 修改或删除

如需修改：
- 任务计划程序 → 任务计划程序库
- 找到`小雨每日运行`
- 右键 → 属性/禁用/删除

## 日志查看

日志文件位置：
`C:\Users\Admin\.openclaw\workspace\daily-learning.log`
