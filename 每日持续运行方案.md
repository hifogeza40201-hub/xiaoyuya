```powershell

# 创建基本任务
# 名称: 小雨每日运行
# 触发器: 每天 06:00
# 操作: 启动程序
# 程序: C:&#46;\Users\Admin\u0026#46;openclaw\workspace\小雨每日运行脚本.bat
```

### 2. Cron任务（通过OpenClaw）

```json
{
  "cron": {
    "jobs": [
      {
        "name": "小雨每日网关检查",
        "schedule": "0 6 * * *",
        "command": "gateway-check.bat"
      },
      {
        "name": "小雨每日自主学习",
        "schedule": "0 6 * * *",
        "command": "autonomous-mode-start.py"
      }
    ]
  }
}
```

### 3. 手动运行脚本

**文件**: `小雨每日运行脚本.bat`
