# 2026-02-13 学习记录

## 自主学习：命令行与自动化

### 目标1：PowerShell基础学习 ✓ 完成

#### 学习内容
1. **变量与数据类型**
   - PowerShell变量以 `$` 开头
   - 基本类型：string, int, double, bool, array, hashtable
   - 数组操作：索引访问、范围访问、添加元素
   - 哈希表：键值对存储，支持点访问和方括号访问

2. **运算符**
   - 算术：+, -, *, /, %
   - 比较：-eq, -ne, -gt, -lt, -ge, -le
   - 字符串：-like (通配符), -match (正则), -contains, -in
   - 逻辑：-and, -or, -not, -xor
   - 类型：-is, -as
   - 范围：..

3. **流程控制**
   - If-ElseIf-Else 条件判断
   - Switch 多分支（支持通配符和正则）
   - For, Foreach, While, Do-While 循环
   - Break, Continue 控制语句

4. **函数**
   - 基本函数声明和调用
   - 参数声明和默认值
   - 高级参数属性：Mandatory, ValidateSet, ValidateRange, Switch
   - 管道函数：ValueFromPipeline
   - 返回哈希表等多值

5. **管道与常用命令**
   - Where-Object 过滤
   - Select-Object 选择属性
   - Sort-Object 排序
   - Group-Object 分组
   - Measure-Object 统计
   - ForEach-Object 遍历
   - Tee-Object 分支输出

6. **文件系统操作**
   - Get-ChildItem/Set-Location 等位置操作
   - New/Remove/Copy/Move/Rename-Item 文件操作
   - Get/Set/Add-Content 内容操作
   - Split-Path/Join-Path 路径操作
   - 环境变量访问: $env:VARNAME

#### 关键发现
1. PowerShell是面向对象的，管道传递的是.NET对象而非纯文本
2. 命令遵循 动词-名词 命名规范 (Verb-Noun)，如 Get-Process
3. 别名系统让Unix用户和Windows CMD用户都能快速上手
4. Get-Help是获取帮助的最重要命令
