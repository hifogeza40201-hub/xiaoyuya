### 调研结果

#### ✅ TTS（文字转语音）- 完整支持
**功能**：AI说话 → 转成语音发给用户

**三种方案**：
1. **Edge TTS**（推荐，免费）
   - 不需要API Key
   - 支持中文语音 zh-CN-XiaoxiaoNeural
   - 配置简单

2. **OpenAI TTS**（需要API Key）
   - 质量更好
   - 速度快

3. **ElevenLabs**（需要API Key）
   - 最自然
   - 可定制声音

**推荐配置**：
```json
{
  "messages": {
    "tts": {
      "auto": "always",
      "provider": "edge",
      "edge": {
        "enabled": true,
        "voice": "zh-CN-XiaoxiaoNeural",
        "lang": "zh-CN"
      }
    }
  }
}
```

#### ✅ Voice Call 插件 - 电话语音通话
**功能**：AI可以打电话或接电话，语音交流
**需要**：Twilio/Telnyx/Plivo 账号
